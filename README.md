# 📊 Digital Marketing & Website Analysis Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard built using a Digital Marketing dataset. It provides insights into website traffic, user engagement, marketing campaign performance, conversions, revenue, and geographic trends. The dashboard helps businesses make data-driven marketing decisions.

---

## 🎯 Objectives
- Analyze website traffic performance.
- Monitor user engagement.
- Evaluate marketing campaign effectiveness.
- Track conversion and revenue.
- Identify top-performing traffic sources.
- Discover geographical performance trends.

---

## 🛠️ Tools Used
- Power BI
- Microsoft Excel / CSV
- DAX
- Power Query

## 📌 Key KPIs

- Total Users
- New Users
- Returning Users
- Sessions
- Conversion Rate
- Revenue
- Bounce Rate
- Session Duration
- Campaign Performance

---

## 📈 Key Insights

- Organic Search generated the highest number of users.
- Paid campaigns delivered the highest revenue.
- Returning users showed a higher conversion rate than new users.
- Mobile devices generated the majority of website traffic.
- Certain regions contributed significantly more revenue than others.
